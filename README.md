# Rubens Matos - Portfolio Landing Page

Landing page estatica, otimizada para SEO tecnico, performance e publicacao no GitHub Pages.

## Estrutura

- `index.html`: pagina principal
- `404.html`: pagina de erro
- `assets/css/styles.css`: estilos
- `assets/js/main.js`: interacoes leves
- `assets/img/*`: imagens e placeholders
- `robots.txt`, `sitemap.xml`, `site.webmanifest`: SEO e indexacao

## Atualizacoes rapidas

1. Substitua `assets/img/profile-placeholder.svg` pela sua foto real.
2. Atualize os textos de apresentacao em `index.html`.
3. Troque os links dos projetos (`Projeto` e `Repositorio`) na secao `#projetos`.
4. Ajuste seus contatos reais (`mailto`, LinkedIn, GitHub) nos metadados e CTAs.

## Deploy no GitHub Pages

Para usar URL principal `https://rubensmatos.github.io/`, o repositorio precisa se chamar exatamente:

`rubensmatos.github.io`

Com esse nome, basta subir os arquivos na branch `main` que o Pages publica automaticamente.
